# A primeira parede

## Arquivos RC

Os arquivos RC (Run Commands ou Runtime Configuration) são arquivos de configuração que armazenam preferências e configurações para ferramentas, aplicações ou ambientes específicos.

O nome vem dos primeiros sistemas Unix, onde "rc" significava "run commands" - comandos que deveriam ser executados na inicialização de um programa.
