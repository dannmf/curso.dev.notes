# Diferença entre Git e Github

## Analogia Principal

Confundir Git e Github é como confundir um arquivo video.mp4 que está em sua máquina com o Youtube.

## Youtube

O Youtube é o local aonde podemos pegar os vídeos que estão no nosso computador e fazer o upload, aonde o Youtube vai hospedar, distribuir e ao redor do conteúdo criar uma rede social/comunidade onde pessoas podem comentar, curtir e compartilhar os vídeos.

## Github

Seguindo o mesmo principio, o github é como o Youtube, porém ele serve para armazenar e agrupar os seus projetos, aonde as pessoas podem fazer o upload, baixar, comentar, curtir e compartilhar os projetos.

O seu Github acaba se tornando uma VITRINE do Dev, aonde você pode exibir os seus projetos, trabalhos etc.

## Git

O Git é um sistema de controle de versão distribuído que permite você manipular os arquivos do seu projeto no seu computador.

Voltando a analogia, o git é onde podemos manipular os nossos "videos.mp4" antes de realizar o upload para o Youtube.
